
# Juatz MVP

## What this is
A minimal executable backend for Juatz:
- Post local requests
- View active requests
- Foundation for trust & response loop

## How to run
1. Install Python 3.10+
2. pip install -r requirements.txt
3. python app.py
4. Open http://localhost:8000

## Endpoints
- GET /
- POST /request
- GET /requests
